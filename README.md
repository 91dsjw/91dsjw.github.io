# 💔 [遗憾向：《未寄出的明信片》](https://91dsjw.github.io/)

苏言在整理旧物时，翻出了一沓未寄出的明信片，每张上面都写着同一个地址，收件人是“安晓”。

那是七年前的夏天，他在丽江的青旅遇见安晓。她扎着马尾，穿着洗得发白的牛仔裤，在院子里弹吉他，唱《南方姑娘》。苏言坐在她对面，听着她的歌声，突然就觉得，丽江的阳光都变得温柔了。 他们一起在束河古镇的石桥上看日落，一起在玉龙雪山脚下堆雪人，一起在深夜的酒吧里，喝着风花雪月啤酒，聊彼此的梦想。安晓说她想环游世界，苏言说他想当一名摄影师，把全世界的风景都拍下来。

离开丽江的前一天，苏言买了一沓明信片，想每天给她寄一张，告诉她他遇见的风景。可回到城市后，他却迟迟没有寄出。他总觉得，等他变得更好，等他有了足够的能力，再站在她面前，告诉她他的心意。 可后来，他从朋友那里听说，安晓在一次旅行中出了意外，永远留在了她最热爱的路上。苏言抱着那沓明信片，在深夜里哭了很久。他终于明白，有些心意，不能等，有些错过，就是一辈子。

现在苏言成了知名的摄影师，他的镜头里有全世界的风景，可唯独没有那个在丽江弹吉他的姑娘。那些未寄出的明信片，成了他心底最深的遗憾。原来爱情里最痛的，不是不爱，是我还在等，你却已经走远。
